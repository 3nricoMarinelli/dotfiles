# Backend Domain - Security & DevOps Knowledge

Domain-specific guidance for Security-DevOps agent working on backend projects.

## Security Considerations

### Authentication
- OAuth 2.0 / OpenID Connect
- JWT validation
- API keys
- Session management
- MFA/2FA

### Authorization
- RBAC (Role-Based Access Control)
- ABAC (Attribute-Based)
- Principle of least privilege
- Permission inheritance

### Network Security
- TLS/SSL
- API gateways
- WAF (Web Application Firewall)
- DDoS protection

### Data Security
- Encryption at rest
- Secrets management (Vault)
- Input validation
- SQL injection prevention
- XSS prevention

### Compliance
- SOC 2
- GDPR
- PCI-DSS
- HIPAA
- ISO 27001
